Introduction

Rule

1. 在合适的世界线发布， 工业界(产业界)内容发布于World，学术界内容发布于Beta，询问发布于Alpha
2. 禁止发布恐吓，诽谤，反政府，恐怖主义，色情，歧视等具备争议性，负面或不法内容
3. 禁止发布广告，非相关资讯(如：菜市场的苹果10块钱3斤)
4. 个人展示的内容请自觉添加前缀：「SHOW:」
5. 发布信息时请填写正确的url信息
6. 用户有完善军规的权利

「本平台对以上条款拥有最终解释权」

Instruction

1. 主体功能均在导航栏，用户可以submit一条新闻，检索新闻（parse 新闻的title和intro）
2. 在Menu菜单，用户可以设置账户-setting、管理发布与收藏、查看统计信息、进行圆桌会议
3. 在主页新闻列表，每当你点开一个手风琴表单，里面会有一副图片(可选)，一段介绍（可选）以及四个按钮， 这四个按钮分别是点赞，打开新闻源链接，对新闻进行讨论和收藏
NOTE:每个功能都经过思考和精心设计，用户凭借直觉就能操作。其中最重要的功能分别是：发布，检索，打开链接和评论。使用评论需要挂载VPN

QA

1. 为什么点赞不可逆？答：点赞功能十分重要，决定了新闻的排序， 所以其被隐藏在手风琴内部，这就大大减少了因为手滑错点的情况，设计被简化，从而无需逆操作
2. 为什么不能踩？答：「恶意规避」
3. 怎么取消收藏？答：在dashboard页面
4. 有没有私信功能？答：后台其实已经实现，但是经过研究发现私信使用率极低属于鸡肋功能，可能不会面世
5. 如何举报违规信息？答：In progress
